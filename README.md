# 📌 Project Overview

This project is an end-to-end **Business Intelligence & Customer Analytics** solution built using **MySQL** and **Power BI**. Using the Brazilian Olist E-Commerce dataset containing **100,000+ transactions**, the project analyzes customer behavior, revenue trends, seller performance, product insights, and operational KPIs through advanced SQL queries and interactive dashboards.

---

# 🎯 Objectives

- Analyze business performance
- Understand customer purchasing behavior
- Identify top-performing sellers
- Evaluate product category performance
- Track monthly revenue trends
- Generate business KPIs
- Build an executive Power BI dashboard

---

# 📂 Dataset

**Dataset:** Brazilian E-Commerce Public Dataset by Olist

Dataset includes:

- Customers
- Orders
- Order Items
- Sellers
- Products
- Payments
- Reviews
- Geolocation

📥 Dataset Source

https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
