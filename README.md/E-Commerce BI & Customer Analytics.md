# **E-Commerce Business Intelligence \& Customer Analytics**



**Project Overview**

This project analyzes the Brazilian Olist E-Commerce dataset using MySQL and Power BI. The goal was to uncover customer behavior, seller performance, sales trends, and business insights through SQL analysis and interactive dashboards.



**Dataset**

Dataset: Brazilian E-Commerce Public Dataset by Olist (Kaggle)



The dataset contains:

\- Customers

\- Orders

\- Products

\- Sellers

\- Payments

\- Reviews

\- Order Items



Total records analyzed: 100,000+ transactions.



**Tools Used**

\- MySQL

\- Power BI

\- SQL

\- DAX
